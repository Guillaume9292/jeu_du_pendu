Le jeu du pendu que j'ai réalisé possède les caractéristiques suivantes :

- Vous devez trouver un mot aléatoire parmi une liste. Chaque mot possède au maximum 7 lettres.

- Lors de chaque partie, vous avez un total de 7 vies et pouvez donc vous tromper de lettre 7 fois au maximum, à défaut la partie est perdue.

- Lors de chaque erreur, le pendu se forme petit à petit jusqu'à prendre sa forme finale au bout de 7 erreurs.

- Si vous n'avez plus de vies, le clavier disparaît et il n'est plus possible de jouer dans cette partie.

- En bas de l'écran, vous avez deux boutons. Le bouton "Nouvelle partie" qui vous propose de lancer une nouvelle session de jeu en vous demandant une confirmation. Le seocnd bouton "Proposer un mot" vous offre l'opportunité de tenter votre chance en cours de partie et tenter de gagner si vous estimez avoir trouvé le mot sélectionné par le pendu.

- La charte graphique repose sur trois couleurs : #1f426c, #e9ecf5 et #ffffff.

- Lors du survol d'une lettre, celle-ci apparaît avec un effet scale + couleur #f55e71.

- Pour réaliser ce jeu du pendu, je me suis aidé du code réalisé par Charlotte sur le forum Discord car j'ai encore des difficultés avec certaines parties du code (relier les lettres cliquées avec le mot à trouver.
